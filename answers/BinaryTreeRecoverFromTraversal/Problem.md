# Recover Binary Tree from Traversal

Given the pre-order and in-order traversal of a binary tree.
Output the post-order traversal of the tree.

# Input
First line N, the number of nodes in the tree. \
Second line and third line, each contains N numbers. \
The second line is the pre-order traversal of the tree. \
The thirs line is the in-order traversal.

# Output
One line of N numbers, the post-order traversal of the tree.

## Example

### Input
4
1 2 4 3 \
2 4 1 3 

### Output
4 2 3 1
